# Orator SSL Proxy

> SSL-terminating reverse proxy with host-based routing, local CA, and Let's Encrypt support

- Host-based HTTPS dispatch with exact, wildcard, and default fall-through
- Trusted local CA — install once, every leaf cert is automatically trusted
- Let's Encrypt automation via HTTP-01 with staging by default
- WebSocket upgrade forwarding on the same dispatch axis
- First-class Fable service provider with zero-config defaults
